# Deep Learning Laboratory

This repository contains all the lab assignments and experiments for the Deep Learning Laboratory course 2026-27 at Shiv Nadar University.

## Repository Structure

The project is structured on a weekly basis. Each folder corresponds to a specific week's assignment and operates as a fully self-contained project with its own data, codebase, and instructions.

### Assignments

- **`1-single_layer_perceptron/`**: Week 1 - Implementation of a Single Layer Perceptron from scratch for binary classification using the Banknote Authentication Dataset.
- *(Future weekly assignments will be appended here in their respective numbered folders)*

## Standard Folder Layout

Inside each assignment folder, you will typically find:

- **`*.ipynb`**: The primary Jupyter Notebook containing the code, analysis, and experiments.
- **`data/`**: The dataset(s) required for the experiment.
- **`images/`**: Visualizations and plots generated by the code.
- **`requirements.txt`**: Python dependencies specific to that assignment.
- **`README.md`**: A concise guide detailing how to execute the code for that week.

## Getting Started

To run an experiment, simply navigate into the specific week's directory, install the required packages using its `requirements.txt`, and follow its local `README.md` execution instructions.
