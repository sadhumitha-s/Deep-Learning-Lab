# Single Layer Perceptron (SLP) for Binary Classification

This repository contains the implementation of a **Single Layer Perceptron (SLP)** from scratch for binary classification using the **Banknote Authentication Dataset**. The project also includes an additional implementation of the perceptron learning algorithm for fundamental logic gates (OR, AND, and NOT), along with the corresponding laboratory report and manual.

## Repository Structure

```
.
├── data/
│   └── data_banknote_authentication.txt
├── images/
│   └── Generated plots for the Banknote dataset
├── images_ad/
│   └── Generated plots for the logic gate experiments
├── logic_gates.ipynb
├── slp.ipynb
├── report.pdf
├── requirements.txt
└── README.md
```

### Contents

* **`slp.ipynb`**

  * Exploratory Data Analysis (EDA)
  * Data preprocessing and feature normalization
  * Single Layer Perceptron implementation from scratch
  * Model training and evaluation
  * Comparison with Scikit-learn's Perceptron
  * Learning rate analysis
  * Performance metrics and visualizations

* **`logic_gates.ipynb`**

  * Perceptron implementation for OR, AND, and NOT logic gates
  * Step-by-step weight updates during training
  * Decision boundary visualization after each weight update (OR and AND)
  * Final learned model visualization for the NOT gate

* **`data/`**

  * Banknote Authentication Dataset used for binary classification.

* **`images/`**

  * Automatically generated visualizations from `slp.ipynb`, including:

    * Feature histograms
    * Correlation heatmap
    * Scatter plot
    * Feature boxplots
    * Training errors vs. epochs
    * Confusion matrix
    * Learning rate comparison

* **`images_ad/`**

  * Visualizations generated by `logic_gates.ipynb`, including intermediate and final decision boundaries for the logic gate experiments.

* **`report.pdf`**

  * Complete laboratory report for Experiment 1.

* **`requirements.txt`**

  * Python package dependencies required to run the notebooks.

## Running the Project

### Single Layer Perceptron

1. Ensure the dataset is available at:

   ```
   data/data_banknote_authentication.txt
   ```

2. Launch Jupyter Notebook:

   ```bash
   jupyter notebook slp.ipynb
   ```

3. Run all notebook cells sequentially.

4. The generated visualizations will be saved automatically in the `images/` directory.

---

### Logic Gate Experiments

Launch the notebook:

```bash
jupyter notebook logic_gates.ipynb
```

Run all cells to generate:

* Weight update logs
* Decision boundary plots for OR and AND gates
* Final classification plot for the NOT gate

The generated figures will be saved in the `images_ad/` directory.

## Dataset

**Banknote Authentication Dataset**

* **Source:** UCI Machine Learning Repository
* **Instances:** 1372
* **Features:** 4 numerical features
* **Classes:** 2 (Authentic / Forged)
